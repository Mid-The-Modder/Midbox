The Filler Update

The Filler Update, or Midbox V0.2, was the standard content-adding update.
This update adds a few features and is the period of time where Midbox would begin to rise.

New Content

Unisons

‣ Added a unison.
The new unison is Corrupt.

‣ Began work on making more instruments unison-compatible.
It didn't make it past the functionality phase, so only the UI exists. It can only change the pitch of some instruments and is completely defunct on the FM.

Chipwaves

‣ Added 2 chipwaves.
The new chipwaves are: Bass-y and Strange.

Envelopes

‣ Added 3 envelopes.
The new envelopes are: Modbox Click, Modbox Bow, and Slow Swell.
In the current version of Midbox, the "Slow Swell" envelope has been renamed to "Swell 5".

‣ Made the "Unison" envelope target available to custom chip instruments.

Keybinds

‣ Added a keybind.
The new keybind is 'P', which sends you to the song player.

Presets

‣ Added a preset category.
The new category is Midbox Noise Presets.

‣ Added 12 presets. They were added as the following:
|
[Midbox Misc Presets (11)] - Odd Snare, Otherworldly Kick, Bit-Crushed Piano, Boop Pad, Spectrum Pad, '3, 2, 1, GO! Pulse', Ambient Pulse 1, Saturn Pad, Build-Up Pad, Ambient Pulse 2, and Hyper Synth Saw.
|
[Midbox Noise Presets (1)] - Hollow Retro Riser.
In the current version of Midbox, the "3, 2, 1, GO! Pulse" was renamed to "Countdown Pulse".

Misc

‣ Added 17 FM operator frequencies.
The new frequencies are: ~0.50x, ~0.75x, 1.50x, 2.50x, ~4x, ~8x, 10x, 12x, 14x, 15x, ~16x, 17x, 18x, 19x, 24x, 32x, and ~32x.

Bug Fixing

Unisons

‣ Fixed an issue where the Hyper, Peak, Deep Shift, Vary, and Energetic unisons wouldn't work on picked string instruments.

Changes

Envelopes

‣ Got rid of the distortion envelope target since it wasn't functional.

‣ Made the "Modbox Blip" envelope more accurate.

Limit Breaking

‣ Changed the detune slider to go from (-200)-200 to (-600)-600 in cents.

‣ Changed the tempo limits to go from 30-320 to 1-750.

‣ Changed the max amount of bars in a song from 256 to 512.

‣ Changed the limits for the minimum/maximum amount of beats per bar from 3-16 to 1-12.

Misc

‣ Added the new features to the random generated instruments as well as tweaked values so most things are weighted equally.

‣ Changed the volume of the 'heavy saw' chipwave and the volumes of the Hyper, Peak, Deep Shift, Vary, Energetic, and Hold unisons.

‣ Edited the text of many tip prompts and the mod's description to be more accurate/have more info.
Also linked a keybind list document in the description, which in current versions of Midbox, has become a part of Midbox.

‣ Replaced various mentions of Jummbox with Midbox.

‣ Changed various properties of new blank songs.
|
The "Unnamed" song name has been renamed to "Untitled Project", loop length increased from 4 to 8, maximum amount of patterns in a channel increased from 8 to 32, and channel amounts from 3 pitch and 1 drum to 4 pitch, 2 drum, and 1 mod.
